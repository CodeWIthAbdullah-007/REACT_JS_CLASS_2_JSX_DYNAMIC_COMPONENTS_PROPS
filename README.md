# REACT_JS_CLASS_2_JSX_DYNAMIC_COMPONENTS_PROPS
# React JS Class 2 - JSX, Dynamic Components & Props 🚀

## 📌 Project Overview
This project is designed to help learners understand the core concepts of React including JSX, dynamic components, and props. It focuses on building reusable components and passing data efficiently between them.

---

## 🎯 Key Features
- ⚛️ JSX syntax usage
- 🔄 Dynamic component rendering
- 📦 Props passing between components
- 🧩 Reusable UI components
- 🖥️ Clean and modular structure

---

## 🛠️ Technologies Used
- React JS ⚛️
- Vite ⚡
- JavaScript (ES6+) 📜
- HTML & CSS 🎨
